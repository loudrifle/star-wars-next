# Document Heading

This is a [Next.js](https://nextjs.org) project

## Requirements

- Node.js 22+

## Getting Started

### Pull/Download

First, pull/download the code

### Install

Install dependencies

```bash
npm install
# or
yarn
# or
pnpm install
# or
bun install
```

### Run

Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
